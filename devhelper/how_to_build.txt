COM-библиотека DevHelper предназначена для получения информации
об устройствах, установленных в системе, и управления ими.

Реализует два co-class'а:

DevHelper.DeviceList: список устройств (описания, сервис, и т.д.)
DevHelper.USBSTORHelper: управляет USB-устройствами хранения данных (флэшками
и съемными винчестерами)

Как компилировать и собирать проект:

Для сборки нужны следующие средства:

1) Microsoft Visual C++ (SP 5)
2) Microsoft Platform SDK (можно бесплатно скачать с microsoft.com)
3) Библиотека WTL версии 7.1 или выше (можно бесплатно скачать оттуда же)
4) Файл cfg.h из Windows DDK (поскольку Windows DDK сейчас бесплатно скачать нельзя,
а можно либо заказать, либо получить по подписке MSDN, то этот файл я положил в проект.
Перед компиляцией его надо положить в C:\Program Files\Microsoft SDK\Include)

Список Include-директорий должен быть выстроен в таком порядке:

SDK\Include
VC98\Include
VC98\MFC\Include
VC98\ATL\Include
VC98\WTL\Include

Для сборки нужно отключить _ATL_MIN_CRT и _ATL_STATIC_REGISTRY, и добавить setupapi.lib
в список LIB-файлов.

Пример использования приведен в скрипте StopUSBDevice.vbs


Компиляция из командной строки:

Надо вызвать build.cmd. Перед этим надо удостовериться, что в переменные окружения 
INCLUDE и LIB в секцию, уникальную для пользователя (не системную) прописаны соответствующие 
пути и директории.
