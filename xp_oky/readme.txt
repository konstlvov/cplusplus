Расширенные хранимые процедуры для MS SQL Server.

Проект для C++ Builder 6.0

Как компилировать и устанавливать:

	Компиляция (билд): открыть проект xp_oky.bpr и выбрать из меню "Project -- Build xp_oky"

	Для установки запустить install_xp_oky.cmd, указав в нем правильный путь к исполняемым (Binn)
	файлам SQL Server, а также корректные security credentials для запуска OSQL.EXE


Как использовать:
	Примеры использования находятся в прилагаемых SQL-файлах.
